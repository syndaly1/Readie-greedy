What is a fantasy book?

How this library works

The catalog page supports search, filtering by tag, sorting (e.g. rating descending), and adding new books. The UI consumes data from a real backend API.

About the Team

We are students working on a web project as part of the Web Technologies course. The project is developed using Node.js and Express.js.

Project Goal

The goal of this project is to practice backend fundamentals such as routing, serving static files, and handling user input through forms, while building a clean UI for a small library catalog.
